Principais Formatações

Negrito / Destaque

Antigamente, para deixarmos um termo destacados em negrito, utilizavamos a tag <b>

Atualmente, para destacar um termo em negrito, utilizamos a tag <strong>

Italico / Ênfase

Também, para deixar um termo em itálico, poderiamos utilizar a tag <i>

E para destacarmos um termo em italico, utilizamos a tag <em>

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>

E no outro paragrafo, temos um texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragrafo

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado

Texto inserido

Também podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi inserido depois

Texto sobrescrito

Para inserir coisas do tipo: x2 + 3

Utilizamos a tag <sup> para isso.

Texto subscrito

Para inserir coisas do tipo H20

Utilizamos a tag <sub> para isso.
